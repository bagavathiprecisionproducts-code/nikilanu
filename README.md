💖 Love Journey Website
A beautiful and romantic birthday surprise website created using HTML, CSS, and JavaScript. This project is designed as an interactive love story with animations, music, and a proposal ending 💍

✨ Features
💌 Multiple interactive pages (Click to continue)
💖 Romantic love quotes on each page
🎬 Smooth fade-in animations
❤️ Floating heart animations
🎶 Background music (continuous playback)
💍 Proposal page (YES / NO interaction)
🎁 Final greeting card page
🔄 Restart love story option
📁 Project Structure
project-folder/
│
├── index.html
├── love.mp3
🚀 How to Run
Download or clone the repository

Make sure both files are in the same folder:

index.html
love.mp3
Open using:

VS Code Live Server (Recommended)
OR double click index.html
🌐 Live Demo (GitHub Pages)
To deploy on GitHub Pages:

Upload files to your repository

Go to Settings → Pages

Select:

Source: main branch
Folder: / (root)
Click Save

Your site will be live at:

https://your-username.github.io/repository-name/
⚠️ Important Notes
Keep love.mp3 in the same folder as index.html

Music will start only after user interaction (browser policy)

If music doesn't play:

Check file name
Check browser sound settings
